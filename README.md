# ToDo-IA-ms

API REST de gerenciamento de tarefas (MVP). Spring Boot 2.7 / Java 11.

## Documentação da API

Com a aplicação rodando (`./mvnw spring-boot:run`):

- Swagger UI: <http://localhost:8080/swagger-ui.html>
- OpenAPI JSON: <http://localhost:8080/v3/api-docs>

Endpoints documentados sob a tag **Tarefas**. Os esquemas dos DTOs (request, response, erro padronizado) são exibidos com descrição e exemplo realista.

> README estruturado completo (Stack, Pré-requisitos, Como rodar, Como testar, Estrutura de pastas) é entrega do `RT-013` no `docs/backlog.md`.
